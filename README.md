# Multi-Purpose Fitting tool
[<img src="./logo.png" width="250"/>](./logo.png)

Done for the Summer Astronomy Space Program, this is a useful notebook for cube fitting designed for JWST. For now, we used Spitzer IRS cubes.

## Requirements
* astropy==5.1
* lmfit==1.0.3 
* dust-extinction==1.1
* specutils==1.7
* matplotlib==3.5.2
* dill==0.3.5.1
